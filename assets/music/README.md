# Music Assets

Place royalty-free MP3 tracks here. Filename format:

    <mood>_<anything>.mp3

## Supported moods
- `motivational_` — hustle/business energy (e.g. motivational_rise.mp3)
- `chill_`        — lo-fi/relaxed vibes
- `energy_`       — hype/fast-paced
- `uplifting_`    — feel-good/inspirational
- `cinematic_`    — dramatic/storytelling

## Where to get free tracks
- https://pixabay.com/music/         (CC0, no attribution needed)
- https://freemusicarchive.org/      (check license per track)
- https://incompetech.com/           (Kevin MacLeod - CC BY)
- https://www.bensound.com/          (free tier available)

## Example file list
```
motivational_rise.mp3
motivational_grind.mp3
chill_lofi1.mp3
energy_hype.mp3
uplifting_bright.mp3
cinematic_dark.mp3
```

The system automatically loops tracks to match video duration,
fades in (1s) and out (2s). Music volume is 10% so speech stays clear.
